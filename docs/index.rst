Bienvenue dans la documentation de CODE_BIDS
============================================

Ce dépôt rassemble des scripts d'automatisation pour préparer des données de neuro-imagerie
selon la spécification BIDS. Les modules principaux sont écrits en Python et sont complétés
par des scripts Shell et Julia utilisés pour reconstruire, normaliser ou analyser les données
issues d'acquisitions Bruker.

La documentation fournie ici est prête pour une publication sur `Read the Docs <https://readthedocs.org/>`_.
Elle présente les points d'entrée et les scripts clés afin de faciliter la découverte du
code.
